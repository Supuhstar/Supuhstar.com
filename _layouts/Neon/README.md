# Neon-Jekyll-Layouts
Some layouts for Jekyll to make it easier to implement Neon


# Dependencies #
This requires the following to be used as submodules in the same repo where this is included:
- https://github.com/BlueHuskyStudios/Neon-Jekyll-Basics     (1.x) must be in the folder `_includes/Neon-Jekyll-Basics`
- https://github.com/BlueHuskyStudios/Neon-Jekyll-Components (1.x) must be in the folder `_includes/Neon-Jekyll-Components`
